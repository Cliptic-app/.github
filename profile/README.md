# Cliptic

**Cliptic** is a self-hosted clipboard manager and syncing platform
designed to keep your clipboard history accessible across all your devices.

---

## ✨ Features

* 📋 Clipboard history manager (desktop)
* 🔄 Cross-device syncing via self-hosted server
* 🔒 Privacy-first — Full end to end encryption
* ⚡ Fast, lightweight, and minimal
* 🖥️ Desktop client (Windows, Linux, Mac is also possible, contributions are welcome)
* 📱 Android client (coming soon)

## 🧭 Vision

Cliptic aims to be a simple, reliable alternative to cloud-based clipboard tools. Combined with local clipboard managers like [Ditto](https://github.com/sabrogden/Ditto)
